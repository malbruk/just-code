Features

כל מה שהופך את Claude Code למצוין — בפאנל אחד

Just Code בנוי על ה-Claude Agent SDK הרשמי של Anthropic, כך שהאג׳נט שרץ בפאנל הוא בדיוק האג׳נט שמריץ את Claude Code בטרמינל.

העוצמה של Claude Code, אותה רמה

אותה מערכת כלים מלאה — עריכת קבצים עם דיפים בעורך, הרצת פקודות, חיפוש, שרתי MCP — ואותם מודלים: Fable, Opus, Sonnet ו-Haiku, עם חשיבה מורחבת ורמות מאמץ.

מתחברים עם המנוי הקיים שלכם

התחברות עם מנוי Claude (Pro/Max) דרך חשבון claude.ai — בלי חיוב נוסף — או עם API Key מה-Console. בחירה אחת בהגדרות, וזהו.

תמיכה מלאה ב-RTL

עברית ואנגלית חיות יחד באותה שיחה: כיווניות אוטומטית לכל פסקה, קוד נשאר LTR, והטקסט תמיד קריא — כמו שהדף הזה מדגים.

רק קוד — באמת

השיחה מסוננת לפיתוח בלבד: שאלות שאינן קוד או טכנולוגיה נדחות בנימוס, כולל ניסיונות לעטוף אותן בבקשה "טכנית". כלי עבודה, לא חבר לשיחה.

ועוד בפנים: אישורי הרשאות לכל כלי היסטוריית שיחות אזכורי ‎@קבצים פקודות ‎/ מצב תכנון מעקב שימוש ומכסות צ׳אט בסייד-בר או בטאב

Requirements

מה צריך כדי להתחיל

שתי דקות של הכנה, ואפשר לעבוד.

  • מנוי Claude — או חשבון ליצירת API Key
    מנוי Pro/Max ב-claude.ai (מומלץ), או חשבון Anthropic Console ליצירת מפתח API בחיוב לפי שימוש.
  • Claude Code מותקן במחשב
    npm install -g @anthropic-ai/claude-code — התוסף מאתר את ההתקנה אוטומטית.
  • VS Code 1.90 ומעלה
    מתקינים מה-Marketplace, נכנסים עם החשבון — ומתחילים לקודד.
